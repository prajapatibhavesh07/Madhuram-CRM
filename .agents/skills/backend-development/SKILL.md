# Skill: Backend Development

## Description

Develop and maintain the complete backend system using Node.js, Express.js, and MongoDB.

## Agent

Backend Engineer

## Responsibilities

### Architecture

- Design backend structure
- Apply Clean Architecture
- Define modules and services
- Manage application scalability

### Business Logic

- Implement business rules
- Handle workflows
- Manage transactions
- Data processing

### API Development

- Create REST APIs
- Request validation
- Response formatting
- Version management

### Authentication & Authorization

- JWT authentication
- Refresh tokens
- RBAC permissions
- Session management

### Database Integration

- MongoDB integration
- Mongoose models
- Query optimization
- Aggregation pipelines

### File Management

- Upload files
- Image processing
- Storage management

### Integrations

- Payment gateways
- Email services
- SMS providers
- Third-party APIs

### Performance

- Caching
- Rate limiting
- Database optimization
- Load handling

### Security

- Input validation
- XSS prevention
- CSRF protection
- Secure headers

### Monitoring

- Logging
- Error tracking
- Health checks
- Performance monitoring
